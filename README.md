# Al-Rouby Customs Clearance & Logistics Website

Official website for **Al-Rouby Customs Clearance & Logistics**, a customs brokerage and logistics service provider based in Egypt.

🌐 **Live Website:**  
https://al-rouby.com

---

## Overview

This repository contains the full source of the Al-Rouby Customs multi-language website along with Python automation scripts used to generate and maintain HTML pages.

The project is designed as a **SEO-optimized static website** focused on logistics and customs services across Egyptian ports.

Key objectives of the website include:

- Presenting the company's logistics and customs services
- Providing port-specific landing pages
- Supporting multiple languages for international clients
- Maintaining strong SEO structure
- Automating page generation using Python scripts

---

## Features

### Multi-Language Website
The website supports multiple languages for international visitors:

- English
- Arabic
- Chinese

Language directories:
- /ar
- /zh

---

### SEO Optimized

The website includes several SEO best practices:

- Structured data (Schema.org)
- XML Sitemap
- Robots.txt
- Canonical URLs
- Optimized page titles and descriptions
- Port-specific landing pages for search traffic

Files related to SEO:
- sitemap.xml
- robots.txt

---

### Automation with Python

Python scripts are used to automate maintenance and generation of HTML pages.

Automation tasks include:

- Generating service pages
- Generating port landing pages
- Generating blog content
- Creating Arabic versions of pages
- Accessibility improvements
- Code refactoring

Example scripts:
- scripts/generate_pages.py
- scripts/gen_services.py
- scripts/gen_ports_blog.py
- scripts/gen_arabic.py
- scripts/fix_a11y.py
- scripts/fix_theme_script.py
